# 🍬 Sweet Rain

A fun browser game where it rains sweets and you have to eat them by clicking or tapping!

## 🎮 How to Play

1. Click **Rain!** to start the sweet storm
2. Move your mouth cursor around the screen
3. **Click or tap** on falling sweets to eat them
4. Earn 🪙 coins for each sweet you eat
5. Build combos for bonus points!
6. Click **Stop** to pause the rain

## 🍭 Sweet Filter

Toggle which sweets fall using the emoji buttons in the top bar:

| Sweet | Points |
|-------|--------|
| 🍭 Lollipop | 1 coin |
| 🍬 Candy | 1 coin |
| 🍫 Chocolate | 2 coins |
| 🍦 Ice Cream | 3 coins |
| 🍩 Donut | 2 coins |
| 🎂 Cake | 5 coins |

## ⚡ Combo System

Eat sweets quickly to build combos:

- x3 combo → **×2 points**
- x5 combo → **×3 points**
- x10 combo → **×4 points 🔥**

## 🚀 Deployment

### GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your game will be live at `https://yourusername.github.io/sweet-rain/`

### Local

Just open `index.html` in any modern browser — no build step needed!
