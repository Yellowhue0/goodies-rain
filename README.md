# 🍬 Goodies Rain

A fun browser game where it rains goodies and you have to eat them by clicking or tapping! Level up as you earn more points for increasingly challenging gameplay!

## 🎮 How to Play

1. Click **Rain!** to start the goodie storm
2. Move your mouth cursor around the screen
3. **Click or tap** on falling goodies to eat them
4. Earn 🪙 coins for each goodie you eat
5. Build combos for bonus points!
6. Reach score milestones to level up — gameplay gets faster!
7. Click **Stop** to pause the rain

## 🍭 Goodie Types

Pick and mix style candies! Toggle which goodies fall using the emoji buttons in the top bar:

| Goodie | Points |
|--------|--------|
| 🍭 Lollipop | 1 coin |
| 🍬 Candy | 1 coin |
| 🍫 Chocolate | 2 coins |
| 🍦 Ice Cream | 3 coins |
| 🍩 Donut | 2 coins |
| 🐻 Gummy Bear | 2 coins |

## ⚡ Combo System

Eat goodies quickly to build combos:

- x3 combo → **×2 points**
- x5 combo → **×3 points**
- x10 combo → **×4 points 🔥**

## 📈 Level System

Reach score milestones to unlock new levels with faster-falling goodies:

- **Level 1**: 0–99 coins
- **Level 2**: 100–249 coins
- **Level 3**: 250–499 coins
- **Level 4**: 500–799 coins
- **Level 5**: 800+ coins

Each level increases spawn rate for more challenge and faster gameplay!

## 🚀 Deployment

### GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your game will be live at `https://yourusername.github.io/goodies-rain/`

### Local

Just open `index.html` in any modern browser — no build step needed!
